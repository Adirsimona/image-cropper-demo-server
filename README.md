# Project Title
image-cropper-demo-server, server side implementation with Node js.


## Getting Started

### Installing

* Clone this project.
    ```
    git clone https://github.com/Adirsimona/image-cropper-demo-server.git
    ```
Then:
    ```
    cd image-cropper-demo-server
    ```
Then run:
    ```
    npm i
    ```

### Executing program

```
node app.js
```
